🔧 Login Implementation Notes

Passwordless Auth Strategy

Technical Flow
1. Client Request:
Call Supabase Auth: supabase.auth.signInWithOtp()
2. Redirect Configuration:
Must set emailRedirectTo to /portal or /portal/dashboard.
3. Resend Integration:
Supabase SMTP settings must point to Resend API to ensure delivery to JP carriers.
Bilingual Email Templates
Ensure the magic link email itself is bilingual or matches the user's browser language detected at the time of sending.
User Experience
Mobile Hand-off: The link must ideally open the app internally if PWA is installed, or handle the cross-app transition cleanly.
