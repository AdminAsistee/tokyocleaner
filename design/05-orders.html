🔧 Order Schema & Statuses

Data Model for the View Layer

Order Status Enum
scheduled: New booking from Zapier.
confirmed: Cleaner assigned in AppSheet.
in_progress: Cleaner is on-site.
completed: Finished + Photos uploaded.
Query Logic
Upcoming: WHERE date >= today AND status != 'completed'
Past: WHERE date < today OR status = 'completed'
